Same-frame editing → temporal fixer

The frozen image editor is run on each frame independently ("same-frame editing"). Because each frame is edited on its own, the result flickers — the temporal inconsistency our fixer removes. Our fixer is a causal video model trained only on clean-video reconstruction (no edited data); editing transfers for free. The image editor is Qwen-Image-Edit (4-step distilled, cfg=2.0). Note: the "Δ" numbers are a simple frame-to-frame mean-pixel-difference proxy for temporal inconsistency (not a standard metric, and blur-confounded); rigorous evaluation uses flow-warped warp-error + VBench. Every demo uses all 33 consecutive frames in their original order, at the clean source's native FPS; there is no temporal subsampling.
