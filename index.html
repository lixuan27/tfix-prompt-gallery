Temporal_Fixer — Reconstruction Prompt Gallery

Every reconstruction prompt we tried, with its two best reconstruction examples (left = clean source, right = the frozen editor's per-frame reconstruction). Editor = Qwen-Image-Edit-2509 int4 + Lightning-v2 (nunchaku), 4-step, full denoising from Gaussian noise per frame — never tuned. The fixer (a separate causal video model) is trained only to undo the temporal inconsistency you see on the right; at test the only thing that changes is the editor's prompt (reconstruction → edit). 76 reconstruction clips across 38 prompt variants.

Metrics per variant: Δstr = injected frame-to-frame structural inconsistency; driving Δstr = same on driving footage; frame-corr = across-frame consistency of the injected change (lower = more per-frame re-interpretation, the behavior we want).
